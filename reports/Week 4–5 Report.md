## Objective
The objective of this project was to transition from image classification to real-time video understanding by implementing an object detection and multi-object tracking pipeline. The project focused on detecting pedestrians using YOLOv8 and tracking their trajectories across consecutive video frames using the SORT algorithm.

---
## Dataset

A real-world street video containing multiple pedestrians was used to evaluate both object detection and tracking performance.
The video was processed frame-by-frame, enabling real-time detection and continuous identity assignment for moving objects.

---
## Part I — Object Detection using YOLOv8

### Model
The Ultralytics implementation of **YOLOv8 Nano (YOLOv8n)** was used as the object detector.
Only the **Person** class was considered during inference to simplify the tracking pipeline.
### Detection Pipeline

The following steps were performed:

- Load pretrained YOLOv8n weights
- Perform inference on each video frame
- Filter detections using confidence thresholding
- Extract bounding box coordinates
- Visualize detections on the output video

The raw detector outputs, including bounding box coordinates, confidence scores, and predicted classes, were also analysed to better understand the YOLO prediction format.

---
## Part II — Multi-Object Tracking using SORT

### Tracking Pipeline
The detections generated by YOLOv8 were used as inputs to the SORT tracker.

The tracking pipeline consisted of:

- Kalman Filter for motion prediction
- Hungarian Algorithm for optimal assignment
- IoU-based matching between predicted and detected bounding boxes
- Track creation and deletion for object lifecycle management

Each detected pedestrian was assigned a unique tracking ID, enabling identity preservation across multiple frames.

## Training Configuration

|Component|Configuration|
|---|---|
|Detector|YOLOv8 Nano (Pretrained)|
|Target Class|Person|
|Framework|Ultralytics YOLO|
|Tracker|SORT|
|Motion Model|Kalman Filter|
|Data Association|Hungarian Algorithm|
|Matching Metric|Intersection over Union (IoU)|

## Results
The combined YOLOv8 and SORT pipeline successfully performed real-time pedestrian detection and tracking on street video sequences.

The pretrained YOLOv8 detector provided reliable person detections, while the SORT tracker maintained consistent object identities across consecutive frames using motion prediction and data association. The integration demonstrated the effectiveness of the tracking-by-detection paradigm for real-time video analysis.
